* idea中的maven

```
q ：用idea创建maven项目上时发现骨架无法生成

a ：从网上找了许多篇博客说在maven的runner中加上-DarchetypeCatalog=local就行了，使用本地的骨架，但是如果本地骨架没
有的话，始终无法创建maven项目所以我们要先下载好archetype-catalog.xml文件放入到你的本地maven仓库中去，这样问题就解决了。

```
